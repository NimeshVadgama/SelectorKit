Query Views with CSS like Selectors

Selector Kit provides clean and strong way to select views through selectors. This is like querying HTML elements with jQuery using CSS selectors. This can be used for testing, but there is no Apple's private (hidden) APIs used and may be shipped with your production code (at your own lisk).

Syntax

selector ::= selector component
           | component

component ::= raw_component | '<' raw_component '>'

raw_component ::= E | E quantifiers | E quantifiers identifier | E identifier
                | identifier

E ::= view-class-name
    | view-class-name
    | '*'

quantifiers ::= quantifiers quantifier
              | ε

quantifier ::= '[' attribute_quantifiers ']'
             | ':' pseudo_class_quantifier

attribute_quantifiers ::= attribute_quantifiers ',' attribute_quantifier
                        | attribute_quantifier

attribute_quantifier ::= attribute_name attribute_relation attribute_value
                       | attribute_name

attribute_name ::= identifier | string

attribute_relation ::= '=' | '^=' | '$=' | '*='

attribute_value ::= identifier | string | number

pseudo_class_quantifier ::= 'first-child'
                          | 'last-child'
                          | 'nth-child(' number ')'
                          | 'nth-of-type(' number ')'
                          | 'marked(' string ')'
                          | 'tagged(' number ')'

identifier ::= '#' identifier
             | '#' string

Examples

UITableViewCell#'hello-button' UILabel:first-child[text='hello world']
UITableViewCell#"author's name" UILable:nth-child(3)
#items-table <:UITableViewCell UILabel[isHidden]

Semantics (excerpt)

The method - (NSArray*)select:(UIView*) of SelectorKit returns array of views such that c... c ⊦ view... view.

** Sequence **

c... ⊦ view...    c ⊦ view
-----------------------------
c1... c2... c ⊦ view... view

** Single component **

E ⊦ view   quantifier_1 ⊦ view  ...  quantifier_n ⊦ view  identifier ⊦ view
--------------------------------------------------------------------------
E quantifier_1 .. quantifier_n identifier ⊦ view

** Exact class **

[NSStringFromClass([view class]) isEqualToString:view-class-name]
------------------------------------------------------------------
'<' view-class-name '>' ⊦ view

** Subclass **

[view isKindOfClass:NSClassFromString(view-class-name)]
--------------------------------------------------------
view-class-name ⊦ view

** Wildcard **

'*' ⊦ view

** Attribute existence **

view.name != nil
-----------------
[name] ⊦ view

** Comparison **

[view.name isEqualToString:value]
----------------------------------
[name=value] ⊦ view

** Identifier **

[view.accessibilityIdentifier isEqualToString:identifier]
----------------------------------------------------------
# identifier ⊦ view
